# 🌐 IoT Automation Projects

Welcome to the IoT Automation Projects repository by Circuit Craft Lab.

This repository contains Internet of Things (IoT) projects built using Arduino, ESP8266, ESP32, sensors, relays, and wireless communication technologies. The goal is to demonstrate practical automation systems for learning, experimentation, and real-world applications.

## 🚀 Features

- Smart Home Automation
- Wireless Device Control
- Sensor Monitoring Systems
- IoT-Based Alert Systems
- Real-Time Data Collection
- Arduino & ESP Programming Examples

## 🛠️ Hardware Used

- Arduino Uno / Nano
- ESP8266 NodeMCU
- ESP32
- Ultrasonic Sensor (HC-SR04)
- IR Sensor
- Servo Motor
- Relay Module
- DHT11/DHT22
- LEDs and Buzzers

## 📖 Getting Started

1. Clone the repository
2. Open the desired project folder
3. Upload the Arduino sketch to your board
4. Make the required circuit connections
5. Test and modify the project as needed
